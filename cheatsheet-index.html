HTML Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<th> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
<colspan> Column span Defines how many columns a td element should span.
<rowspan> Rowspan Defines how many rows a td element should span.

HTML Form Tags

Tag Name Description
<form> Form The element is used to collect and send information to an external source.
<input> Input The element is used to render a variety of input fields on a webpage.
<label> Label The element is used to identify a specific <input> element with label.
<select> Select The element is used to create an empty dropdown list.
<option> Option The element is used to create a dropdonw list's choice.
<datalist> Data List The element is used to implement a basic search/autocomplete functionality for a <input> element (value of id must match with <input>'s list attribute).
<textarea> Text Area The element is used to create a text-box for multi-line input.

Form Attributes

Attribute Name Description
action Action Defines where form sends the information.
method Method Defines what form does with the information once it's sent.
type Type Defines how form gets rendered to a page.
name Name Defines the key-value pair that will keep the data sent by form.
value Value Defines the key-value pair that will keep the data sent by form; if set, form initialy renders that value until user digits something else.
id Identifier Defines the element's identifier for linking and styling purpose.
for For Defines which <label> element labels.
list List Defines which <datalist> element considers.
rows Rows Defines how many rows has <textarea> element.
cols Columns Defines how many cols has <textarea> element.
step Step Defines how much increment or decrement (with arrows) the user's entered number.
min Minimum Defines the lower bound limit.
max Maximum Defines the upper bound limit.

Form Input Types

Type Name Description
text Text Defines a free plain text area.
text Plain Text Defines a free plain text area.
password Obscured Text Defines a free obscured text area.
number Number Defines a field that could keep only digits.
range Slider Defines a slider that needs lower and upper bounds and the increase-step.
checkbox Check Box Button Defines a checkbox choice that needs label to be understood.
radio Radio Button Defines a set of 'only one selectable choice' choices that needs labels to be understood.
submit Submit Button Defines a submit button to send user's entered data where <form>'s action attribute points.
